Bitácora del Proyecto

En esta sección se documenta el proceso de desarrollo del proyecto, incluyendo decisiones de diseño, dificultades encontradas y cambios importantes realizados.

Decisiones de diseño

  • Se eligió una paleta de colores sencilla para facilitar la lectura y navegación.
  • La estructura del sitio se organizó en secciones claras: Inicio, Acerca de, Contacto y Bitácora.
  • Se utilizó HTML y CSS para asegurar compatibilidad y facilidad de mantenimiento.

Dificultades encontradas y soluciones

  • Problemas con la integración del menú principal: se resolvió utilizando listas y enlaces relativos.
  • Dificultad en la adaptación a dispositivos móviles: se implementó diseño responsivo en CSS.

Cambios importantes durante la implementación

  • Se reorganizó la estructura de carpetas para mejorar la gestión de archivos.
  • Se actualizó el menú principal para incluir la sección de Bitácora.
